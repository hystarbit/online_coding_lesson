인간의 언어로 설명하는 SERVER
존맛닭갈비
닭갈비 2인분 주세요
SERVER: 서빙하는 사람

서버: 요구를 하면 가져다줌
스테이크를 요청하면 스테이크를 갖다주고
저거를 요청하면 저거를 그대로 갖다주고

네이버 SERVER에게 요청을 해보겠다

네이버: 갖다줌(응답)
나: 만화 페이지 요첨
네이버: 갖다줌(응답)

서버: 요청을 받으면 요청한 내용 보내주는 프로그램

서버: 닭갈비집 알바생
서버개발자: 닭갈비집 알바생 메이커

네이버웹툰 서버개발자가 짜는 코드
어떤사람이 comic.naver.com으로 접속하면
네이버웹툰 메인 html파일을 전송해주셈

서버: 요청을 받으면 요청한 내용을 보내주는 프로그램

요청 -> HTTP 요청
요청은 4개 방식이 있음
1. 읽기
2. 쓰기
3. 수정
4. 삭제

1. 읽기(GET) - 뭔가 읽고 싶을 때
나 이런 페이지 일고싶음! 
이라는 요청 가능

2. 쓰기(POST) - 뭔가 생성해주셈
댓글 작성
블로그 포스트 작성
"이거 글좀 올려주세요~"

3. 수정(PUT) - 뭔가 수정할 때

4. 삭제(DELETE) - 뭔가 삭제할 때

사용자는 GET/POST/PUT/DELETE 요청가능
      ↑
서버: 요청을 하면 요청한 내용을 보내주는 프로그램

어떤 사람이 /list라는 페이지를 GET요청하면...
거기 해당하는 list.html파일을 보내줌

Node.js를 이용해서 JavaScript 문법으로 서버짤거임
