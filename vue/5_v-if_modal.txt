(조용히 따라치셈)

동적인 UI 만드는 법:
0. HTML CSS로 디자인해두셈 (기본)
1. UI의 현재 상태를 데이터로 저장해둠
2. 데이터에 따라 UI가 어떻게보일지 작성

동적인 UI 만드는 법:
1. UI의 현재 상태를 데이터로 저장해둠
"모달창이 지금 어떻게 보여야함?"
"서브메뉴가 지금 어떻게 보여야 함?"
"탭이 지금 어떻게 보여야 함?"
"그 UI가 지금 어떻게 보여야함?"

모달창열렸니 : false


2. 데이터에 따라 UI가 어떻게보일지 작성

v-if = "조건식"
조건식이 참일 때만 HTML 보여줌
(밑에 있는 모달창열렸니 data가 true면 보여줄것임

셋팅 이렇게 2-step으로 잘 해놓으면
UI 켜고 끄기 쉽게 가능

동작인 UI 만들면
0. 디자인해두고
